```java
class a {
    public void bubbleSort(int[] source) {
        for(int i = source.length - 1; i > 0; i--) {
            for(int j = 0; j < i; j++) {
                if(a[j] > a[j+1])
                    swap(source, j, j+1); //交换，具体实现略
            }
        }
    }
}
```

选择排序
```java
class a {
    public void selectSort(int[] source) {  
        int min;  
        for(int i = 0; i < source.length; i++) {  
            min = i;  
            for(int j = i + 1; j < source.length; j++) {  
                if(a[j] < a[min])  
                    min = j;    
            }  
            swap(i, min);  
        }  
    }
}
```
